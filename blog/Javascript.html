Javascript Functions vs Ruby Methods

February 15, 2016

In Ruby, a method is a series of statements that perform a specific tasks. A method must start with lower case letters, it can include a _(underscore) or !(exclamation point). All methods start with the keyword, "def" followed by the method name. The next line will include the block of code that will perform the task. Lastly, the method must be completed with the keyword "end"

Ruby Method def add(a,b) a + b end add(5,5) //===> 10

In Javascript, a function is similar to a Ruby method. A function is piecce of program wrapped in a value. Such values can be applied in order to run the wrapped program. You can call a function by putting () after an expression that produces a function value. All functions start with the word "function

Javascript Function function functionName() { console.log("Hello World"); } functionName();

As you can see function and methods are pretty simiar. The major difference has to do with synax! Good luck on your journey to mastering JS.
